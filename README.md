# Digital Farm Labour Record Management System

## Description
A beginner-friendly MySQL DBMS project inspired by real farm activities like onion curing and weed cleaning. The system manages worker records, attendance, wages, and daily labour tasks using SQL queries and relational database concepts.

## Features
- Worker record management
- Attendance tracking
- Wage calculations
- Daily work records
- SQL query operations

## Technologies Used
- MySQL
- SQL
- DBMS
